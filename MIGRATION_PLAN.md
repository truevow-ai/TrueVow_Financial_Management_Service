# Migration Plan: Vite React → Next.js 14 App Router

## Current State
- Vite + React 18 + React Router
- Custom JWT auth
- Custom components
- Axios for API calls

## Target State
- Next.js 14 (App Router)
- Clerk authentication
- shadcn/ui-style components
- Recharts for charts
- Server components by default

## Migration Steps
1. Initialize Next.js 14 project structure
2. Migrate components to Next.js App Router
3. Replace auth with Clerk
4. Convert to shadcn/ui-style components
5. Update API calls for Next.js
6. Add Recharts integration
